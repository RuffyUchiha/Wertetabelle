# Wertetabelle
Dieses Programm berechnet und generiert eine Wertetabelle für eine quadratische Funktion in einem benutzerdefinierten Bereich.
Funktionalitäten:

    Start- und Endwerte festlegen: Der Benutzer gibt die Start- und Endwerte in die entsprechenden Eingabefelder ein.
    Berechnung der Werte: Durch Klicken auf die Schaltfläche "Berechnen" wird die Funktion y = 0.25 * x^2 - 3 für jeden Wert zwischen Start- und Endwert berechnet.
    Anzeige der Ergebnisse: Die berechneten Werte werden in einer Tabelle angezeigt, in der die erste Spalte die x-Werte und die zweite Spalte die entsprechenden y-Werte enthält.
    Programm beenden: Klicken Sie auf die Schaltfläche "Beenden", um das Programm zu schließen.

Verwendung:

    Geben Sie einen Start- und Endwert in die Eingabefelder ein.
    Klicken Sie auf die Schaltfläche "Berechnen", um die Tabelle zu generieren.
    Die Tabelle zeigt die berechneten Werte für die Funktion y = 0.25 * x^2 - 3 an.
    Um das Programm zu beenden, klicken Sie auf die Schaltfläche "Beenden".
